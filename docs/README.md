# 다리 건너기 게임 기능 구현 목록
## 기능 구현 목록
* 입력 기능
  - 다리 길이 입력 기능
  - 사용자가 건널 칸 입력 기능
  - 게임 재시작 여부 입력 기능


* 출력 기능
  - 현재까지 이동한 다리 정보 출력 기능
  - 최종 결과 출력 기능
  - 출력에 필요한 문자열 관리 기능


* 다리 위 아래 설정 기능
  - 랜덤으로 1, 0번호 발생 기능


* 다리 만드는 기능
  - 랜덤으로 설정된 다리 위, 아래 구현 기능


* 다리 진행 결과 출력 기능
  - 다리 기본 틀 기능
  - 다리에 조건과 규칙 기능
  - 입력한 값에 따라 출력되는 기능
  - 최종 게임 결과 출력 기능


* 다리 건너기 구현 기능
  - 칸 이동 기능
  - 재시작 기능


* 게임 진행 기능
